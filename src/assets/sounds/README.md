# Sons de Notificação

Esta pasta contém os arquivos de áudio utilizados para notificações da extensão.

## Arquivos Necessários

- `notification.mp3` - Som padrão para notificações críticas

## Especificações Recomendadas

- Formato: MP3
- Duração: Máximo 2 segundos
- Tamanho: Máximo 100KB
- Bitrate: 128kbps
- Volume: Normalizado

## Como Adicionar

1. Adicione o arquivo `notification.mp3` nesta pasta
2. Certifique-se de que o arquivo seja curto e discreto
3. Teste o volume em diferentes dispositivos

## Alternativa

Se não houver arquivo de som disponível, o sistema funcionará silenciosamente e registrará no console quando tentar reproduzir o som.
